## Project Description
This is a responsive Weather Web Application built with HTML, CSS, and JavaScript. It fetches real-time weather data and 5-day forecast using the **Open-Meteo API** (no API key required).

### Features Implemented:
- Search weather for any city in the world
- Display current weather conditions (temperature, description, humidity, wind speed)
- 5-day weather forecast
- Responsive design (works on mobile and desktop)
- Error handling and loading states
- Clean and modern UI matching project specifications

### Technologies Used:
- HTML5 (Semantic tags)
- CSS3 (Flexbox, Responsive design, Hover effects)
- JavaScript (Async/Await, Fetch API, DOM Manipulation)

### How to Run:
1. Clone or download the repository
2. Open `index.html` in your browser
3. Or view live demo via GitHub Pages

### API Used:
- Geocoding API: `https://geocoding-api.open-meteo.com`
- Weather API: `https://api.open-meteo.com`


**Note**: UV Index is displayed as **N/A** because it is not available in the free Open-Meteo API.
